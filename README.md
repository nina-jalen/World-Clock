## ğŸ”§ Customization

You can easily expand this project by adding more cities to the dropdown menus.

To add a new location, open the index.html file and insert more <option> elements with the correct timezone values.

Example:

```html
<option value="Asia/Tokyo">Tokyo</option>
<option value="Australia/Sydney">Sydney</option>
```

Just use any valid IANA timezone (like Europe/London, America/New_York, etc.) and the app will display the current local time for that selection.
